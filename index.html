Kart Racing Pro Telemetry analyser

JavaScript web tool to analyse the telemetry data generated by the karting simulator Kart Racing Pro.

To run the program press select file and then select the file created by Kart Racing Pro.

The tabs are
Analyse to view graphs of speed, throttle, break, steer etc
Laps to show lap times
About information of the tool

If you hover in the graphs of the analysis tab you can se were on the track the data was recorded. The colored dots show where you shift.

All graphs including the track map is zoomable
